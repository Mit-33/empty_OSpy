# OSpy
This is a little O.S. in python language.

## First use
In your first use you **MUST** do this:
* Rename *empty_OSpy* to *OSpy*
* Run OSpy.exe.py
* Click on the AppStore's logo
* Click on AppStore vX.X (X is the number of the version)
* Press \[escape\]
* Run this command in a terminal in OSpy folder:
```cmd
python OSpy.exe.py AppStore
```
* Then click on OSpy.exe.py in the window

After this you can run normaly OSpy
